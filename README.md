# DecodeLabs Internship Portfolio

A modern personal portfolio website built with plain HTML, CSS, and JavaScript. The site showcases a responsive developer portfolio with custom interactions, motion effects, and 3D visuals.

## Overview

This project is a polished portfolio landing page for a full-stack developer and data scientist. It includes:

- Animated hero section with three.js particle visuals
- Custom cursor and hover interactions
- Mobile-friendly navigation with a responsive hamburger menu
- Animated preloader with Lottie integration
- Multi-section page layout for About, Skills, Projects, Experience, and Contact
- Modern design using CSS variables, fluid typography, and subtle motion

## Key Features

- **Responsive design** for desktop and mobile screens
- **Smooth scroll and section highlighting** for improved navigation
- **Custom mouse cursor** and interactive hover states
- **GSAP scroll animations** for entrance effects
- **Three.js background scene** in the hero section
- **Lottie animations** for preloader and section visuals
- **Accessible markup** with ARIA labels and semantic structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- [Three.js](https://threejs.org/)
- [GSAP](https://greensock.com/gsap/)
- [ScrollTrigger](https://greensock.com/scrolltrigger/)
- [Lottie](https://airbnb.io/lottie/)

## Project Structure

- `index.html` — main markup and page sections
- `style.css` — layout, responsive styling, animations, and theme
- `script.js` — interactive behaviors, preloader, custom cursor, and animations

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/abaid9658/DecodeLabs-internship.git
   ```
2. Open `index.html` in your browser.

## Notes

- This is a static portfolio website, so no backend or database is required.
- All interactions are handled on the client side with JavaScript.
